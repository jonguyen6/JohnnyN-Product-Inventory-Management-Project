# Inventory-Management-Project-Product-DEMO-
GitHub repository containing the necessary files for a simple command-line product management system demo via JDBC and MySQL Workbench. 

# Project Features:
- View all products
- Search for a product via product ID
- Add a new product
- Change a product's current stock/inventory count
- View all products in low stock

# Project Pre-requisites:
Before running the project, make sure you have the following installed:
- Java JDK
- MySQL Server & Workbench
- JDBC

# How to run the project:
1. Run the SQL file in MySQL Workbench. It should already contain lines of code for sample data.
2. Open the java files in your IDE of choice, and then update the Database connection credentials in DataBaseConnection.java if needed.
3. Compile and run the main java file.
